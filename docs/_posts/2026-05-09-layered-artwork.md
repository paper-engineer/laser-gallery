An experiment. I think it turned out quite well. 

Material: 3mm pine ply

Time: 10 min, not including design

focus set at 6mm+standard, after the discovery that actually the focus tool might not be fully correct. cut at 650/100 again, well, it mostly worked. 

![pieces of a star]({{ site.url }}{{ site.baseurl }}/assets/images/star-pieces.jpg)

It came out and I put it together wrong the first time - for some reason I hadn't quite thought ahead to multiple possible orientations. So the next thing I'll be doing with this file is probably making it square-square so that the pieces can be rotated any which way for assembly. 

![assembled other way]({{ site.url }}{{ site.baseurl }}/assets/images/star-assembled-1.jpg)
